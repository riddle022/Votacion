# Cellshop Duty Free - Setup Instructions

## Criar Usuário Admin

Para acessar o painel administrativo, você precisa criar um usuário admin no Supabase:

1. Acesse o Supabase Dashboard: https://pfnuyoomanalazzmvkpl.supabase.co
2. Vá para Authentication > Users
3. Clique em "Add User"
4. Insira o email e senha do administrador
5. Clique em "Create User"

## Acessar o Painel Admin

1. Na aplicação, clique no botão "Admin" no canto inferior direito
2. Faça login com o email e senha criados
3. Agora você pode gerenciar as opções de votação e visualizar os resultados

## Funcionalidades do Admin

### Opções de Votação
- Adicionar novas opções em 3 idiomas (PT, ES, EN)
- Editar opções existentes
- Ativar/Desativar opções
- Excluir opções

### Resultados
- Visualizar total de votos
- Ver gráficos com porcentagem de cada opção
- Filtrar por período de datas
- Exportar dados para CSV
